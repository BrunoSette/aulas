# Exercicios

## 1. Dobrar valores
```javascript

  // your code here
}

console.log(doubleNumbers([2, 5, 100])); // [4, 10, 200]
```

## 2. Transformar números em string
```javascript

  // your code here
}

console.log(stringItUp([2, 5, 100])); // ["2", "5", "100"]
```
